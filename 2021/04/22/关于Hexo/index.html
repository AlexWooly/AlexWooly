关于Hexo

Hexo介绍

hexo是一个快速、简洁且高效的博客框架,它支持几条简短的命令发布博客,因此是对于初学者比较友好的博客框架。

安装Hexo

  • 安装Node.js
  • 安装Git
  • 安装Hexo npm install -g hexo-cli
  • 执行npx hexo

  • hexo init blog 创建个人博客（链接外网会很卡）

  • git clone git@gitee.com:b-match/blog.git 使用镜像

  • 进入博客目录 cd blog

  • npm install hexo-deployer-git –save 在文件夹内安装，新文件夹要重新安装

  • 修改配置文件 _config.yml vim _config.yml

  • 在deploy中输入配置

Hexo 提交

  • Hexo clean 清除缓存
  • Hexo generate 生成新的静态文件 简写为 hexo g
  • Hexo deploy 部署到博客 hero d
  • 去仓库的Gitee Pages点击更新/github不需要
